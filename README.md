# NYC Schools Analysis

## Project Overview

This project analyzes NYC high school data including SAT results, incidents, and school information.

**Author**: Alexander Kuhn  
**Date**: January 2026  
**Program**: Webeet Onboarding Day 5

---

## What This Project Does

- Analyzes SAT test results from 421 NYC high schools
- Cleans and validates data
- Stores data in PostgreSQL database
- Creates visualizations and insights
