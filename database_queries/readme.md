# Database Exploration

## Overview

SQL-based analysis of NYC school data using Python and PostgreSQL.

## Data

- **Database**: PostgreSQL (Neon)
- **Tables**: 
  - `high_school_directory`
  - `school_demographics`
  - `school_safety_report`

## Analysis

1. Database connection setup (psycopg2)
2. School distribution by borough
3. Average percentage of English Language Learners (ELL) per borough
4. Schools with special needs support
5. Accessibility analysis

## Key Insights

- Number of schools per borough
- ELL percentages across boroughs
- Schools with functional accessibility
- Special education support availability

## Files

- `day3_sql_analysis.ipynb` - SQL queries via Python

## Technologies

- Python (pandas, psycopg2)
- PostgreSQL
- SQL queries
